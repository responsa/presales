Come resettare la password della casella PEC

La password della casella PEC non può essere recuperata o ripristinata, ma solo cambiata (reset). Per resettare la password procedere come di seguito descritto:
Non occorre conoscere la password corrente della casella PEC per reimpostarla. 
  1. aprire la pagina Reset password;
  2. inserire la casella PEC (completa del dominio per esempio nomecasella@pec.it, nomecasella@pec.nomedominio.est, postmaster@pec.nomedominio.est);
  3. inserire il codice fiscale o la partita IVA del titolare della casella PEC o del Dominio;
  4. spuntare Non sono un robot per l'inserimento del codice di verifica (captcha);
  5. cliccare Prosegui:
  6. il sistema invia una comunicazione all'email di recupero contenente un link;
    Il link ha una validità di 3 ore e può essere utilizzato una sola volta.  Non può essere richiesto un ulteriore link se non sono trascorse le 3 ore dalla precedente richiesta, trascorse le quali occorre procedere a una nuova richiesta di reset password.
     
    Se l'email di recupero non è corretta o non è più possibile accedervi procedere alla modifica:
    • per il servizio PEC (nomecasella@pec.it) come descritto in questa guida (procedura con invio documentazione);
    • per il servizio PEC su Dominio (nomecasella@pec.nomedominio.est, postmaster@pec.nomedominio.est) come descritto nella guida dedicata.
    Una volta modificata l'email di recupero ripetere la procedura di reset password.​
  7. cliccare sul link ricevuto e alla pagina Reset password che si apre inserire la Nuova password e confermarla nel campo Ripeti password;
  8. cliccare Cambio password per completare la procedura. La nuova password ha una validità immediata:
    La scelta della password deve rispettare tutti i requisiti elencati nella schermata.  Il sistema segnala eventuali errori, come una mancata corrispondenza tra i campi Nuova Password e Ripeti password o il mancato utilizzo dei parametri obbligatori. Per maggiori informazioni consultare la guida completa  Requisiti e suggerimenti per la scelta di una password efficace.
